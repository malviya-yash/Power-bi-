# Power-bi-
This Power BI project provides an interactive sales and performance analysis for a retail Super Store. The dashboard helps visualize key business insights using data on sales, profit, quantity, and customer segments.

Key Objectives

Analyze overall sales and profit trends across regions, categories, and time periods.

Identify top-performing products and customers driving revenue.

Compare regional performance using visual KPIs and filters.

Monitor profitability and discounts to support better pricing decisions.

Features

Dynamic Dashboards: Interactive visuals for category, region, and sub-category analysis.

Time-Based Insights: Monthly and yearly sales performance trends.

KPIs & Metrics: Total Sales, Profit, Quantity Sold, Average Discount.

Drill-Through Reports: Explore data by region, category, or product level.

Clean Data Modeling: Relationships built between Orders, Returns, and Customers tables.
